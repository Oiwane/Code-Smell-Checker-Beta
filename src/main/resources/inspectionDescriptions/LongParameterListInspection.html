Detect Code smell of "Long Parameter List".


This inspection detect methods that are longer than the specified "parameter list length".
Specify the allowable upper limit for "parameter list length".
The "parameter list length" here refers to the number of method arguments.


How to refactor
1. Execute refactoring of "Replace Parameter with Method" if you can get data by querying in this method.
2. Execute refactoring of "Introduce Parameter Object" if you use values as parameter that should be handled together.
3. Execute refactoring of "Preserve Whole Object" if you are using multiple values from an object as parameter.

コードスメル『長すぎるパラメータリスト』を検出します


このインスペクションは指定した『パラメータリストの長さ』より長いパラメータリストを持つメソッドを検出します。
『パラメータリストの長さ』の許容上限を指定してください。
ここでの『パラメータリストの長さ』はメソッドの引数の数を指します。


リファクタリング方法
1.メソッド内で問い合わせてデータを取得できる場合、『メソッドによるパラメータの置き換え』を行ってください。
2.まとめて扱うべき一連の値をパラメータとして使っている場合、『パラメータオブジェクトの導入』を行ってください。
3.あるオブジェクトから複数の値をパラメータとして使っている場合、『オブジェクトそのものの受け渡し』を行ってください。
