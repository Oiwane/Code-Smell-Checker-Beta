Detect Code smell of "Message Chains".


This inspection detect statements that are longer than the specified "length of method chain".
Specify the allowable upper limit for "length of method chain".
The "length of method chain" here refers to the number of times an object requests something from another.
(e.g. 3 times for the code "a.getThis().getThat().foo()")


How to refactor
Execute refactoring of "Hide Delegate".

コードスメル『メッセージの連鎖』を検出します


このインスペクションは指定した『メソッドチェーンの長さ』より長い文を検出します。 『メソッドチェーンの長さ』の許容上限を指定してください ここでの『メソッドチェーンの長さ』とはオブジェクトが別のオブジェクトに何かを要求する回数を指します。 (例えば、『a.getThis().getThat().foo()』といったコードの場合は3です。)


リファクタリング方法
『委譲の隠蔽』を行ってください。
